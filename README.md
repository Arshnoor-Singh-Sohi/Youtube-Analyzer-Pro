## 🚀 Quick Start

### ⚡ **5-Minute Setup**

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/youtube-analyzer-pro.git
cd youtube-analyzer-pro

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set up API key
echo "GOOGLE_API_KEY=your_api_key_here" > .env

# 4. Run the application
streamlit run app.py

# 5. Open browser
# Navigate to http://localhost:8501
```

### 🎯 **First Analysis**

1. **Paste YouTube URL**: Any public video with captions
2. **Select Analysis Type**: Choose "Comprehensive" for your first try
3. **Click "🔍 Analyze Video"**: Wait 30-60 seconds for processing
4. **Explore Results**: Navigate through tabs to see different insights
5. **Try Chat**: Ask questions about the video content
6. **Export**: Download your analysis in preferred format

## 🛠️ Detailed Installation Guide

### 📋 **Prerequisites**

#### **System Requirements**
- **Operating System**: Windows 10+, macOS 10.14+, or Linux (Ubuntu 18.04+)
- **Python**: Version 3.8 or higher
- **Memory**: Minimum 4GB RAM (8GB recommended)
- **Storage**: At least 1GB free space
- **Internet**: Stable connection for API calls

#### **Required Accounts**
- **Google Cloud Account**: For Gemini API access
- **YouTube Account**: For accessing video metadata (optional)


```bash
# Solution: Install Python 3.8+
# Windows: Download from python.org

# Linux: sudo apt update && sudo apt install python3.9
```

**Issue: Permission denied**
```bash
# Solution: Use virtual environment
python -m venv env
source env/bin/activate  # Linux/macOS
env\Scripts\activate     # Windows
```

**Issue: API key not working**
```bash
# Solution: Verify API key
# 1. Check Google AI Studio for correct key
# 2. Ensure .env file format is correct
# 3. Restart application after changing .env
```

**Issue: Dependencies not installing**
```bash
# Solution: Update pip and try alternative sources
python -m pip install --upgrade pip
pip install -r requirements.txt --index-url https://pypi.org/simple/
```

**Issue: Streamlit not found**
```bash
# Solution: Install Streamlit explicitly
pip install streamlit>=1.28.0
streamlit hello  # Test installation
```

### 🔄 **Processing Pipeline**

<details>
<summary><b>📊 Analysis Workflow</b></summary>

**Phase 1: Input Processing (5-10 seconds)**
1. URL validation and format checking
2. Video ID extraction
3. Metadata retrieval (title, duration, thumbnail)
4. Availability verification (captions, accessibility)

**Phase 2: Content Extraction (10-20 seconds)**
1. Transcript download via YouTube Transcript API
2. Text cleaning and preprocessing
3. Segment timing and structure analysis
4. Language detection and validation

**Phase 3: AI Analysis (30-60 seconds)**
1. Summary generation based on selected type
2. Key takeaway extraction
3. Important quote identification
4. Action item generation
5. Topic and theme analysis
6. Sentiment analysis
7. Timeline construction
8. Q&A pair extraction

**Phase 4: Post-Processing (5-10 seconds)**
1. Result formatting and structuring
2. Session data preparation
3. Chat context initialization
4. Export preparation

**Phase 5: Display and Interaction**
1. Results presentation in tabbed interface
2. Chat interface activation
3. Export options preparation
4. Session storage completion

</details>

```python
        body = f"""

        """

        if export_data:
            attachment = MIMEBase('application', 'octet-stream')
            attachment.set_payload(export_data)
            encoders.encode_base64(attachment)

            )
            msg.attach(attachment)
        
        # Send email
        try:
            server = smtplib.SMTP(self.smtp_config['host'], self.smtp_config['port'])
            server.starttls()
            server.login(self.smtp_config['username'], self.smtp_config['password'])
            server.sendmail(msg['From'], msg['To'], msg.as_string())
            server.quit()
            return True
        except Exception as e:
            print(f"Email sending failed: {e}")

        self.Base = declarative_base()
        
        # Define Analysis model
        class Analysis(self.Base):
            __tablename__ = "analyses"
            
            id = Column(Integer, primary_key=True)
            video_id = Column(String(50), unique=True)
            video_title = Column(String(500))
            channel = Column(String(200))
            duration = Column(String(20))
            summary_type = Column(String(50))
            analysis_data = Column(Text)  # JSON data
            created_at = Column(DateTime)
            user_session = Column(String(100))
        
        self.Analysis = Analysis
        self.Base.metadata.create_all(self.engine)
    
    def save_analysis(self, analysis_data):
        """Save analysis to database"""
        session = self.SessionLocal()
        try:
            analysis = self.Analysis(
                video_id=analysis_data['video_info']['video_id'],
                video_title=analysis_data['video_info']['title'],
                channel=analysis_data['video_info']['channel'],
                duration=analysis_data['video_info']['duration'],
                summary_type=analysis_data['settings']['summary_type'],
                analysis_data=json.dumps(analysis_data),
                created_at=datetime.now(),
                user_session=analysis_data.get('session_id', 'unknown')
            )
            session.add(analysis)
            session.commit()
            return analysis.id
        except Exception as e:
            session.rollback()
            print(f"Database save failed: {e}")
            return None
        finally:
            session.close()
    
    def get_analytics(self):
        """Get usage analytics"""
        session = self.SessionLocal()
        try:
            from sqlalchemy import func
            
            stats = session.query(
                func.count(self.Analysis.id).label('total_analyses'),
                func.count(func.distinct(self.Analysis.video_id)).label('unique_videos'),
                func.count(func.distinct(self.Analysis.user_session)).label('unique_users'),
                self.Analysis.summary_type,
                func.count(self.Analysis.summary_type).label('type_count')
            ).group_by(self.Analysis.summary_type).all()
            
            return {
                'total_analyses': stats[0].total_analyses if stats else 0,
                'unique_videos': stats[0].unique_videos if stats else 0,
                'unique_users': stats[0].unique_users if stats else 0,
                'summary_type_distribution': {stat.summary_type: stat.type_count for stat in stats}
            }
        finally:
            session.close()
```
